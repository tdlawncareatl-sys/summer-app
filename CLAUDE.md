# Summer Friend Group Coordination App

## What It Is
A lightweight web app for ~12 friends to coordinate summer plans. Handles availability blackouts, event date voting, and a shared idea hub.

## Core Rule
Every decision should reduce friction, not add process. Mobile-first always. Simple and working beats big and broken.

## Stack
- **Framework:** Next.js (App Router)
- **Database:** Supabase
- **Styling:** Tailwind CSS
- **Hosting:** Vercel

## Database Tables
- `users`
- `availability`
- `events`
- `date_options`
- `votes`
- `ideas`

## Voting Logic
| Response | Points |
|----------|--------|
| Best     | 3 pts  |
| Works    | 1 pt   |
| No       | 0 pts  |
